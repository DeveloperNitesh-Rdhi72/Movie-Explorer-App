# Movie Explorer App

A dynamic web application that allows users to search and explore movies in real time.
The app fetches movie data from a public API and displays details such as posters, ratings, and movie information instantly.

This project demonstrates working with APIs, asynchronous JavaScript, and dynamic DOM manipulation.
